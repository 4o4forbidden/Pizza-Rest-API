# Pizza API
Pizza API is a Rest API to order / manage PIZZAS <3 

## Clone the project
```
git clone this repo ^_^
```

## Project Directory
```
go to the project directory and run the following commands in your terminal
```

## Install packages
```
npm install
```

## Run API

### Install Nodemon
```
npm install -g nodemon
```
```
npm start
```
